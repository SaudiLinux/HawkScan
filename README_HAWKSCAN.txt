# HawkScan - أداة فحص أمان المواقع

تم تطويره بواسطة: Saudi Linux
الإصدار: 1.0.0

## الوصف

HawkScan هي أداة قوية مكتوبة بلغة Python لفحص أمان المواقع واكتشاف نقاط الضعف المحتملة. تقوم الأداة بإجراء مجموعة متنوعة من الفحوصات الأمنية للمواقع المستهدفة، بما في ذلك فحص سجلات DNS، وجمع معلومات Whois، وفحص رؤوس HTTP، وفحص شهادات SSL، وفحص المنافذ المفتوحة.

## المميزات

- فحص سجلات DNS (A, AAAA, MX, NS, TXT, SOA, CNAME)
- جمع معلومات Whois
- فحص رؤوس HTTP وتحليل رؤوس الأمان
- فحص شهادات SSL
- فحص المنافذ الشائعة المفتوحة
- إمكانية حفظ النتائج بتنسيق JSON
- واجهة سطر أوامر سهلة الاستخدام
- دعم التنفيذ المتوازي للفحوصات

## المتطلبات

- Python 3.6 أو أحدث
- المكتبات المذكورة في ملف requirements_hawkscan.txt

## التثبيت

1. تأكد من تثبيت Python 3.6 أو أحدث على جهازك
2. قم بتثبيت المكتبات المطلوبة باستخدام الأمر التالي:

```
pip install -r requirements_hawkscan.txt
```

## الاستخدام

```
python hawkscan.py -t example.com [الخيارات]
```

### الخيارات المتاحة

- `-t, --target`: الهدف المراد فحصه (مثال: example.com) [مطلوب]
- `-o, --output`: مجلد حفظ النتائج
- `--threads`: عدد العمليات المتزامنة (الافتراضي: 10)
- `--timeout`: مهلة الاتصال بالثواني (الافتراضي: 30)
- `-v, --verbose`: عرض معلومات تفصيلية
- `--dns`: فحص سجلات DNS
- `--whois`: جمع معلومات Whois
- `--headers`: فحص رؤوس HTTP
- `--ssl`: فحص شهادة SSL
- `--ports`: فحص المنافذ المفتوحة
- `--all`: تنفيذ جميع الفحوصات

## أمثلة للاستخدام

### إجراء جميع الفحوصات على موقع

```
python hawkscan.py -t example.com --all -o results
```

### فحص سجلات DNS ورؤوس HTTP فقط

```
python hawkscan.py -t example.com --dns --headers -o results
```

### فحص شهادة SSL والمنافذ المفتوحة مع زيادة عدد العمليات المتزامنة

```
python hawkscan.py -t example.com --ssl --ports --threads 20 -o results
```
